## 필수 Plotly 차트 (8개)

1. 유동성 환경 점수 (게이지)
2. 자금 흐름 (막대)
3. 섹터별 내부자 매수 비중 (파이/도넛)
4. 종합 투자 적합도 (게이지)
5. 포트폴리오 배분 (트리맵, `branchvalues: total` 필수)
6. 시간별 비중 조정 (라인, stackgroup)
7. 스마트머니 vs 리테일 (양방향 막대)
8. 지역 ETF YTD 성과 (막대)

## 핵심 작성 원칙

### Tier 분류 기준

| Tier | 기준 | 확신도 |
|------|------|--------|
| 1 | 내부자(A등급) + 애널리스트(STRUCTURAL) + 자금유입 3개 수렴 | ⭐⭐⭐⭐⭐ |
| 2 | 2개 수렴 + 섹터 트렌드, 내부자 B+ | ⭐⭐⭐⭐ |
| 3 | 1개 강한 신호 + 잠재력 | ⭐⭐⭐ |
| 4 | 특수 상황 (고위험-고수익) | ⭐⭐ |

### 필수 필터 (반드시 적용)

- **내부자 시그널 등급 연동**: A등급 → Tier 1 후보, C 이하 → Tier 3~4 또는 제외, F → 완전 제외
- **"가짜 업사이드" 차단**: 래그 비율 > 5x → 제외, 2~5x → 경고+1단계 하향
- **EPS 지속성 필터**: STRUCTURAL → 유지, ONE-TIME → 1단계 하향
- **10b5-1 매수 하향**: 10b5-1은 시그널 강도 낮음
- **순매도 종목 제외**: 다른 임원 매도 > 매수면 제외

### 교차 시그널 심층 분석

- 5개 보고서 간 **방향이 바뀌거나 상충하는 신호** 2~4개를 테이블로 정리
- 각 괴리에 **낙관적/비관적 양면 해석 + 확인 조건** 필수
- 스마트머니 vs 리테일 괴리 수준: [정상/확대/극단적]
- 옵션: SPX P/C vs Equity P/C 괴리, VIX 기간구조, 0DTE 비중

### 신규 시스템 리스크

- 과거 사이클에 없던 **새로운 유형** 리스크만 기재 (1~4개)
- 각 항목에 "왜 새로운가" + 정량 임계점 필수
