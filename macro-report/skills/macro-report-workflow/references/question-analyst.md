# 애널리스트 목표가 변동 — 질문 템플릿

> macro-scanner에게 전달되는 데이터 수집 가이드

## 수집 범위

- 기간: 현재일 기준 최근 1개월
- 대상: 월스트리트 주요 애널리스트 목표가 변동

## 수집 항목 (15개)

1. **3명+ 동시 상향 종목** — 기관명, 애널리스트명, 이전/신규 PT, 상향폭, 등급, 날짜
2. **종목별 상향 폭 + 컨센서스 분산도** — 평균 상향률 + 최고~최저 PT 범위 (범위가 넓을수록 의견 분열)
3. **상향 이유의 질 구분** — EPS 추정치 상향 vs P/E 배수 확장 반드시 구별. EPS 상향의 지속성도 구분: STRUCTURAL(구조적, 3분기+ 지속) vs ONE-TIME(일회성) vs MIXED
4. **현재가-목표가 괴리율** — 상승 여력 (%)
5. **시가총액 구간별 분포** — 메가캡/대형/중형/소형 상향·하향 분포
6. **섹터별 목표가 트렌드** — 상향/하향 집중 섹터와 배경
7. **목표가 하향 집중 종목** — 하향 전 2주간 공매도 변화 확인 (선행 공매도 = 정보 비대칭). CEO 교체 트리거 여부 확인
8. **주요 기관별 패턴 + 정확도** — Goldman, JPM, MS 등 기관별 변화 + TipRanks 기준 12개월 정확도. 배치 업그레이드(같은 주 여러 종목 변경) = 섹터 뷰 전환 시그널
9. **목표가 달성률 통계** — 과거 유사 조건(3개 기관+, 20%+ 여력)에서 실제 달성까지 평균 기간
10. **내부자 매수 동시 종목** — 애널리스트 상향 + 내부자 매수 수렴
11. **옵션 시장 선행 신호** — 목표가 발표 전 2주 콜옵션 거래량, P/C Ratio, IV 변화
12. **공급망 "뒤처진 종목"** — 대형주 대폭 상향의 공급망 시총 $10B 이하 소형주 중 목표가 미반영 종목
13. **컨센서스 초과 종목** — 주가 > 컨센서스인 종목. "따라잡기" vs "과열 반전" 판단
14. **컨센서스 하향 래그** — 주가 하락률 / 컨센서스 하향률 = 래그 비율. 5x 이상이면 "가짜 업사이드"
15. **섹터별 Revision Ratio** — (상향 건수 - 하향 건수) / 전체 건수. 이전 기간 대비 변화율 포함
