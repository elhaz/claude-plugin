# Consumer & Industrial Sector Metrics

## Retail

**Representative Companies:** WMT, COST, HD, TGT

| Metric | Description | Good Level |
|--------|-------------|------------|
| **Same-Store Sales (SSS)** | Existing store revenue growth | 3%+ healthy |
| **Inventory Turnover** | Inventory efficiency | Higher is better |
| **Gross Margin** | Gross profit margin | Stable or improving |
| **Operating Margin** | Operating profit margin | Depends on model |
| **Sales per Sq Ft** | Store productivity | Compare to peers |

### Key Considerations
- SSS is the core metric for retail health
- E-commerce impact on traditional retailers
- Inventory management crucial for profitability

## E-commerce

**Representative Companies:** AMZN, SHOP, MELI, SE

| Metric | Description | Good Level |
|--------|-------------|------------|
| **GMV (Gross Merchandise Value)** | Total transaction value | Growth rate focus |
| **Take Rate** | Platform fee rate | Stable maintenance |
| **CAC (Customer Acquisition Cost)** | New customer cost | Lower is more efficient |
| **LTV/CAC** | Lifetime value ratio | 3x+ excellent |
| **Fulfillment Cost %** | Logistics as % of revenue | Improving trend |

### Key Considerations
- Network effects and scale advantages
- Marketplace vs 1P mix affects margins
- Logistics investment payback period

## Aerospace & Defense

**Representative Companies:** BA, LMT, RTX, NOC, GD

| Metric | Description | Good Level |
|--------|-------------|------------|
| **Backlog** | Unfilled orders | Multi-year revenue visibility |
| **Book-to-Bill** | Orders / Revenue ratio | 1.0+ healthy |
| **Operating Margin** | Profitability | 10%+ healthy |
| **FCF Conversion** | FCF / Net Income | 100%+ excellent |
| **Government Contract %** | Defense revenue share | Stability indicator |

### Key Considerations
- Long contract cycles provide visibility
- Program execution risk (cost overruns)
- Defense budget dependent
- Commercial aerospace cyclicality
