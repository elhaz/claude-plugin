# Financial Sector Metrics

## Asset Managers

**Representative Companies:** Blackstone (BX), Apollo (APO), Ares (ARES), KKR, Blue Owl (OWL)

| Metric | Description | Good Level |
|--------|-------------|------------|
| **FRE (Fee-Related Earnings)** | Management fee-based earnings, more stable than GAAP | Higher is better |
| **FRE Margin** | FRE / Management fees, operational efficiency | 50%+ excellent |
| **DE (Distributable Earnings)** | Actual distributable profit | Evaluate with FRE |
| **Permanent Capital %** | Funds with no maturity, revenue stability | 80%+ excellent |
| **Fee-Paying AUM** | AUM generating fees (distinct from total AUM) | Growth trend |
| **Fundraising** | New capital raised, growth indicator | Track quarterly |

### GAAP vs Non-GAAP Notes
- GAAP net income includes non-cash items (stock comp, intangible amortization) causing volatility
- Investors focus on **FRE, DE** as core metrics
- High payout ratios on GAAP basis may be normal if DE coverage is healthy

### Key Considerations
- Permanent capital protects against redemption pressure
- FRE margin indicates operating leverage
- AUM growth vs fee rate compression dynamic

## BDC (Business Development Company)

**Representative Companies:** ARCC, OBDC, MAIN, HTGC, PSEC

| Metric | Description | Good Level |
|--------|-------------|------------|
| **NII (Net Investment Income)** | Net investment income, dividend source | Dividend > NII is risky |
| **NAV (Net Asset Value)** | Net asset value per share | Compare to stock price |
| **NAV Premium/Discount** | Stock price vs NAV | Discount may be opportunity |
| **Dividend Coverage** | NII / Dividend | 100%+ required |
| **Non-Accrual Ratio** | Non-performing loan ratio | Below 5% is healthy |
| **Leverage Ratio** | Debt/Equity | 1.0x-1.25x appropriate |

### GAAP vs Non-GAAP Notes
- GAAP net income includes unrealized gains/losses
- **NII** is key for evaluating dividend sustainability
- NAV can be volatile due to mark-to-market adjustments

### Key Considerations
- NAV discount can signal market distress or opportunity
- Watch for dividend cuts when NII coverage declines
- Leverage limits are regulatory (typically 2:1 max)

## Banks

**Representative Companies:** JPM, BAC, WFC, C, USB, PNC

| Metric | Description | Good Level |
|--------|-------------|------------|
| **NIM (Net Interest Margin)** | Net interest margin, core profitability | 3%+ healthy |
| **CET1 Ratio** | Core equity tier 1, regulatory capital | 10%+ safe |
| **ROE** | Return on equity | 12%+ excellent |
| **Efficiency Ratio** | Cost/Revenue, lower is more efficient | Below 60% excellent |
| **Loan Loss Provision Ratio** | Allowance vs bad loans | Higher is more conservative |
| **Loan-to-Deposit Ratio** | Loans/Deposits, liquidity | 80-90% appropriate |

### Key Considerations
- NIM sensitive to interest rate environment
- CET1 ratio determines capital return capacity
- Credit quality trends more important than current levels

## Insurance

**Representative Companies:** BRK.B, PGR, ALL, TRV, MET, AIG

| Metric | Description | Good Level |
|--------|-------------|------------|
| **Combined Ratio** | Loss ratio + expense ratio, below 100% is profit | Below 95% excellent |
| **Loss Ratio** | Claims paid / Premiums | Lower is better |
| **Book Value Growth** | Net asset growth, long-term value creation | 10%+/year excellent |
| **ROE** | Return on equity | 15%+ excellent |
| **Float** | Investable funds between premium collection and claims | Larger is advantageous |
| **Investment Yield** | Return on investment portfolio | Depends on rate environment |

### Key Considerations
- Combined ratio below 100% means underwriting profit
- Float provides free leverage for investment returns
- Catastrophe exposure can cause earnings volatility
