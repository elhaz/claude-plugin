# 한국 주식 분석 가이드 (KR Market)

한국 주식(KRX KOSPI/KOSDAQ) 분석 시 적용하는 시장 특수 사항.

## 회계 기준: K-IFRS

### 연결 vs 별도 재무제표

한국 기업은 **연결**과 **별도** 재무제표를 모두 공시한다. 미국의 GAAP/Non-GAAP 개념이 아니라, **연결/별도 기준 차이**가 핵심.

| 기준 | 포함 범위 | 용도 |
|------|---------|------|
| **연결** | 모회사 + 자회사 전체 | 그룹 전체 실적 평가 |
| **별도** | 모회사만 | 본업 실적 평가 |

**반드시 양쪽 모두 확인**. 특히 자회사 비중이 큰 지주형 기업(LG화학→LG에너지솔루션, SK이노베이션→SK온 등)은 연결/별도 차이가 매우 큼.

### EPS 주식수 기준

한국 기업 중 **보통주/우선주 분리 상장** 기업이 많다.

| 기준 | 주식수 | 사용처 |
|------|--------|--------|
| 보통주 기준 | 보통주만 | 한국 증권사 리포트 일반적 |
| 총 주식수 | 보통주 + 우선주 | Investing.com, Bloomberg 등 글로벌 소스 |

**EPS 기재 시 어떤 주식수 기준인지 반드시 명시**할 것.

### Non-GAAP EPS 없음

한국에는 미국식 Non-GAAP EPS 개념이 없다. 대신:
- **영업이익**: 한국에서 가장 중요한 수익성 지표 (미국의 EBITDA에 해당하는 위상)
- **지배주주 순이익**: 연결 순이익에서 비지배지분 제외

## 대주주/지배구조 분석 (한국 특수)

한국 재벌 그룹은 **지배구조 할인(Korea Discount)**이 존재한다.

### 필수 확인 항목

| 항목 | 설명 |
|------|------|
| **대주주 지분율** | 오너 일가 + 특수관계인 합산 |
| **자회사 지분가치** | 상장 자회사 지분의 시장가치 |
| **NAV 할인율** | (순자산가치 - 시총) / 순자산가치 |
| **순환출자** | 그룹 내 순환출자 구조 여부 |
| **주주행동주의** | 팰리서캐피탈 등 행동주의 펀드 개입 여부 |

### SOTP(Sum-of-the-Parts) 밸류에이션

자회사 지분가치가 큰 한국 지주형 기업은 **SOTP 방식이 필수**:
1. 상장 자회사 지분가치 (시장가 × 지분율)
2. 비상장 자회사 가치 (P/E 또는 P/B 적용)
3. 본업(별도 기준) 가치
4. 순부채 차감
5. 지배구조 할인 적용 (통상 30~50%)

## 수급 분석

| 주체 | 의미 |
|------|------|
| **외국인** | 순매수/순매도 추세 (가장 중요한 수급 지표) |
| **기관** | 연기금, 보험, 투신 등 |
| **개인** | 대부분 역방향 지표 |
| **공매도** | 한국은 공매도 제한/금지 기간이 있음 (2023~2025 한시적 금지 등) |

## 밸류에이션 특수성

- **P/B**: 한국 시장에서 P/E보다 중요한 경우 많음 (적자 기업, 자산주)
- **PEG**: 한국에서는 거의 사용하지 않음
- **배당**: 연 1회 결산배당이 일반적 (분기배당 도입 중이나 아직 소수)
- **자사주**: 소각보다 보유가 일반적 (지배력 유지 목적)
- **Korea Discount**: 동일 업종 글로벌 Peer 대비 20~40% 할인 거래가 일반적
